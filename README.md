# kosmos

Интерактивная 3D-модель Солнечной системы в браузере. Одна HTML-страница, реальный 3D на three.js, PWA с офлайн-режимом.

**Версия:** 0.1.0 (ранний доступ)
**Автор:** smuxo

## Возможности

- Солнце с мягким свечением и селективным bloom — светится только звезда, планеты остаются чёткими
- 9 планет с текстурами, наклонами осей и честными пропорциями скоростей вращения
- Земля с шейдером день/ночь: ночная сторона показывает огни городов
- Атмосферное свечение по краю у Земли, Венеры и Марса
- Луна, Фобос и Деймос (3D-модели), 4 галилеевых спутника Юпитера
- Кольца Сатурна и Урана с тенью планеты
- Два пояса астероидов из GLB-моделей
- Мягкие тени (PCFSoft) от солнечного света
- Орбиты с градиентным следом позади планеты, пылевой диск эклиптики
- Клик или тап по планете — камера подлетает и открывается карточка с данными
- MSAA-сглаживание, анизотропная фильтрация текстур, ACES tonemapping
- Экран загрузки с прогрессом
- Панель настроек: скорость орбит, скорость вращения, яркость Солнца

## Управление

| Действие | Мышь | Сенсор |
|---|---|---|
| Вращение камеры | ЛКМ + движение | один палец |
| Зум | колесо | щипок |
| Панорама | ПКМ + движение | два пальца |
| Выбор планеты | клик | тап |

## Структура

```
index.html      весь код приложения
sw.js           service worker, кэш v0.1.0
manifest.json   PWA-манифест
models/         asteroidPack.glb, phobos.glb, deimos.glb
images/         текстуры и иконки
```

## Текстуры

В `images/` должны лежать:

`1.jpg 2.jpg 3.jpg 4.jpg` (фон космоса, кубмапа), `sun.jpg`, `mercurymap.jpg`, `mercurybump.jpg`, `venusmap.jpg`, `venus_atmosphere.jpg`, `earth_daymap.jpg`, `earth_nightmap.jpg`, `earth_atmosphere.jpg`, `moonmap.jpg`, `moonbump.jpg`, `marsmap.jpg`, `marsbump.jpg`, `jupiter.jpg`, `jupiterIo.jpg`, `jupiterEuropa.jpg`, `jupiterGanymede.jpg`, `jupiterCallisto.jpg`, `saturnmap.jpg`, `saturn_ring.png`, `uranus.jpg`, `uranus_ring.png`, `neptune.jpg`, `plutomap.jpg`, `icon-192.png`, `icon-512.png`

Текстуры колец — квадратные PNG с прозрачностью (кольцо-«бублик» по центру).

## Запуск

Нужен любой статический сервер (для загрузки моделей и текстур):

```
python3 -m http.server 8080
```

Открыть `http://localhost:8080`. Для GitHub Pages — просто закинуть содержимое в репозиторий, пути относительные.

## PWA

При первом открытии service worker кэширует страницу, модели и текстуры — дальше всё работает офлайн. Приложение можно установить на домашний экран (fullscreen, альбомная ориентация).

При выпуске новой версии поменять константу `VERSION` в `sw.js` — старый кэш удалится автоматически.

## Производительность

Профиль качества выбирается автоматически по типу устройства:

| | Телефон | Десктоп |
|---|---|---|
| MSAA | 4x | 8x |
| Карта теней | 1024 | 2048 |
| Астероиды | 400 + 1200 | 1000 + 3000 |
| Pixel ratio | до 1.5 | до 2 |

## Технологии

three.js 0.160, dat.gui — подключаются через importmap с CDN, сборка не нужна. Загрузка GLB идёт через XHR + `GLTFLoader.parse`, поэтому работает и в WebView-окружениях с перехваченным fetch.
